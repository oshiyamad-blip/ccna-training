## 設問

**Q1.** OSI 参照モデルの第 3 層（ネットワーク層）の主な役割はどれか。

- A. 隣接機器間でのフレーム転送
- B. ネットワーク間の経路選択
- C. アプリケーションデータの暗号化
- D. ビットの電気信号への変換

**Q2.** L2 スイッチがフレームの転送先ポートを決定する際に参照する情報はどれか。

- A. 送信元 IP アドレス
- B. 宛先 IP アドレス
- C. 宛先 MAC アドレス
- D. ポート番号

**Q3.** トランスポート層の PDU の名称はどれか。

- A. フレーム
- B. パケット
- C. セグメント
- D. ビット

**Q4.** カプセル化の順序として正しいものはどれか（送信側・上位層から）。

- A. データ → パケット → セグメント → フレーム
- B. データ → セグメント → パケット → フレーム
- C. データ → フレーム → パケット → セグメント
- D. データ → セグメント → フレーム → パケット

**Q5.** TCP と UDP が動作する OSI の層はどれか。

- A. 第 2 層
- B. 第 3 層
- C. 第 4 層
- D. 第 7 層

**Q6.** データセンターにおいて、すべての Leaf スイッチがすべての Spine スイッチに
接続され、サーバ間通信が常に同じホップ数になるトポロジはどれか。

- A. 3 階層
- B. Collapsed Core
- C. Spine-Leaf
- D. フルメッシュ WAN

**Q7.** PC0 がルータ 2 台を経由して PC1 へ IP パケットを送るとき、原則として
送信元から宛先まで**変化しない**ものはどれか（NAT は使用していない）。

- A. 宛先 MAC アドレス
- B. 送信元 MAC アドレス
- C. 宛先 IP アドレス
- D. L2 ヘッダ全体

**Q8.** ファイアウォールの主な役割はどれか。

- A. MAC アドレスにもとづくフレーム転送
- B. ポリシーにもとづく通信の許可・遮断
- C. 無線クライアントの収容
- D. IP アドレスの自動割り当て

**Q9.** 同一 LAN 内で通信を開始する際、宛先 IP アドレスに対応する MAC アドレスを
解決するために使われるプロトコルはどれか。

- A. ICMP
- B. DNS
- C. ARP
- D. DHCP

**Q10.**（記述）ラボで観察したことを踏まえ、「スイッチは IP アドレスを見ない」と
言える理由を 1〜2 文で説明せよ。

## 解答・解説（翌日公開・講師用）

| 問 | 解答 | 解説 |
|---|---|---|
| Q1 | B | ネットワーク層は IP アドレスによる経路選択（ルーティング）を担う。A は L2、C は L6、D は L1 |
| Q2 | C | スイッチは MAC アドレステーブルと宛先 MAC を突き合わせて転送する。IP やポート番号は参照しない |
| Q3 | C | L4 = セグメント、L3 = パケット、L2 = フレーム、L1 = ビット |
| Q4 | B | 上位から順にヘッダが付く: セグメント（L4）→ パケット（L3）→ フレーム（L2） |
| Q5 | C | TCP / UDP はトランスポート層（第 4 層）。ポート番号で通信を識別する |
| Q6 | C | Spine-Leaf は Leaf→Spine→Leaf の 2 ホップで必ず届くため遅延が予測しやすい |
| Q7 | C | IP アドレスは端点間で不変（NAT 除く）。MAC はルータを通過するたびに書き換わる |
| Q8 | B | ファイアウォールはルール（ポリシー）で通信を検査し許可 / 遮断する |
| Q9 | C | ARP は IP アドレスから MAC アドレスを解決する。ラボで ping の前に流れたのがこれ |
| Q10 | 例 | 「スイッチは L2 ヘッダの宛先 MAC アドレスだけを参照して転送し、L3 ヘッダ（IP）は読まずにそのまま中継するため。シミュレーションモードでも IP ヘッダが変更されないことを確認できた」等、MAC 参照 / IP 不変に触れていれば正解 |

**採点**: 1 問 10 点、70 点未満は翌朝再テスト。Q10 は趣旨が合っていれば 10 点。
