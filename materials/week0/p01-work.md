# Week0 P1 実習手順書: ファイル操作とスクリーンショット

> 配置先: ドキュメント `02_実習手順書 > Week0 > P1`
> 所要時間の目安: 3 時間 ／ 使用ツール: Windows エクスプローラー標準機能

## ゴール

- フォルダとファイルを作成・コピー・名前変更・移動・削除できる
- 操作を間違えたときに元に戻す(Ctrl+Z)ができる
- 画面の一部を切り取ってスクリーンショットを撮り、貼り付けられる
- タイピング練習を始め、明日から毎朝続ける習慣の第一歩を踏み出す

講義で学んだ「ファイル = データのひとまとまり」「フォルダ = ファイルをまとめる
入れ物」を、実際に自分の手で作って動かして体で覚えるのが今日の実習です。

---

## 手順 1: フォルダとファイルを作る(40 分)

1. デスクトップの何もない場所を右クリックする
2. [新規作成] → [フォルダー]をクリックする
3. 名前が青く反転して入力できる状態になっているので、そのまま「練習」と入力し、
   Enter キーで確定する
4. 「練習」フォルダをダブルクリックして中を開く
5. 開いたフォルダの中で右クリック → [新規作成] → [テキストドキュメント]を
   クリックする
6. ファイル名を「memo」と入力し、Enter キーで確定する

ここまでで、「練習」フォルダの中に「memo」というファイルが 1 つある状態になり
ます。講義で学んだ「住所」でいうと、「デスクトップ → 練習 → memo」という場所
です。

## 手順 2: コピーして、名前を変える(25 分)

1. 「memo」ファイルを 1 回クリックして選択する
2. `Ctrl + C` を押してコピーする
3. 同じフォルダの中で、ファイルは何も選ばずに `Ctrl + V` を押す
4. 「memo - コピー」という名前のファイルが増えたことを確認する
5. 増えたコピーを 1 回クリックで選び、`F2` キー(または右クリック → [名前の
   変更])を押す
6. 名前を「memo2」に打ち替えて、Enter キーで確定する

これで、「練習」フォルダの中に「memo」と「memo2」の 2 つのファイルがある状態
になりました。

## 手順 3: サブフォルダを作り、ファイルを移動する(25 分)

1. 「練習」フォルダの中で右クリック → [新規作成] → [フォルダー]をクリックする
2. 名前を「保管」と入力し、Enter キーで確定する
3. 「memo2」を 1 回クリックで選び、`Ctrl + X`(切り取り)を押す
4. 「保管」フォルダをダブルクリックで開く
5. `Ctrl + V` を押して貼り付ける
6. 「保管」フォルダの中に「memo2」が移動してきたことを確認する

「保管」は、講義でいう「フォルダの入れ子」(フォルダの中のフォルダ)の実例です。

## 手順 4: 元に戻す(10 分)

1. 「練習」フォルダの階層(1 つ上の階層)に戻る
2. `Ctrl + Z` を 1 回押す
3. 直前の移動操作が取り消され、「memo2」が「練習」フォルダに戻ってきたことを
   確認する

`Ctrl + Z` は「1 つ前の操作を取り消す」ショートカットキーです。操作を間違えた
ときにまず試すべき操作なので、ここで一度体験しておきます。

## 手順 5: 削除して、復元する(20 分)

1. 「memo2」を 1 回クリックで選ぶ
2. `Delete` キーを押す
3. 「memo2」がフォルダから消え、ごみ箱に移動したことを確認する
4. デスクトップのごみ箱をダブルクリックで開く
5. 「memo2」を右クリック → [元に戻す]をクリックする
6. 「練習」フォルダに戻り、「memo2」が復元されていることを確認する

削除したファイルは、すぐにはパソコンから完全には消えず、ごみ箱に入るだけです。
ごみ箱から元に戻せることを覚えておくと、誤って削除したときに慌てずに済みます。

**📷ここでスクショ**: 「練習」フォルダを開いた状態で、中に「memo」ファイル、
「保管」フォルダ(中に何もない状態)、「memo2」ファイルが見えている画面を撮影
してください。次の手順 6 で撮り方を説明します。

## 手順 6: スクリーンショットを撮る(30 分)

ここまでの操作結果を記録するために、スクリーンショット(画面の一部を画像として
切り取る操作)を撮ります。

1. `Windows キー + Shift + S` を同時に押す
2. 画面が少し暗くなり、マウスカーソルが十字に変わる
3. 記録したい範囲の左上から右下へドラッグして、四角く範囲を囲む
4. マウスのボタンを離すと、その部分の画像がクリップボード(コピーした内容を
   一時的に置いておく場所)にコピーされる
5. 「memo」ファイルをダブルクリックして開く(メモ帳が起動する)
6. メモ帳の中で `Ctrl + V` を押す。メモ帳は文字専用のアプリなので、画像は
   貼り付かない(画面に何も起きなくても正常。壊れていない)。確認できたら
   メモ帳を閉じ、次の 7 に進む
7. 代わりに Backlog の課題コメント欄、またはメールの本文を開き、そこで
   `Ctrl + V` を押す
8. 先ほど手順 6 の 3 で撮ったスクリーンショット(手順 5 の📷で指示した「練習」
   フォルダの画面)が本文中に貼り付けられることを確認する

この「範囲を選んでスクショを撮り、貼り付ける」操作は、本編の課題提出や Backlog
でのやり取りで毎日使います。今日のうちに手が覚えるまで、もう一度練習しておくと
安心です。

## 手順 7: タイピング練習を始める(10 分)

1. タイピング練習サイト(寿司打、e-typing など)をブラウザで開く
2. 両手をホームポジション(左手 A・S・D・F、右手 J・K・L・;)に置く
3. 5 分間、実際に打ってみる
4. 今日の結果を覚えておき、明日から毎朝 10 分、同じ練習を続けることをここで
   宣言する

タイピング練習は、本編でコマンドを打ち込む土台になります。最初は遅くて構いませ
ん。毎朝続けることに意味があります。

## うまくいかないとき

| 症状 | 確認すること |
|---|---|
| フォルダや名前の入力欄が出てこない | 右クリックしたのが「何もない場所」か確認する(アイコンの上で右クリックすると別のメニューが出る) |
| `Ctrl + V` を押しても何も貼り付かない | 直前に `Ctrl + C` または `Ctrl + X` を押し忘れていないか確認する |
| `Windows キー + Shift + S` が反応しない | 他のアプリがそのショートカットを使っていないか確認する。反応しない場合は[スタート] → [Snipping Tool]から起動できる |
| ごみ箱から[元に戻す]が出てこない | ファイルを右クリックしているか(ダブルクリックで開こうとしていないか)確認する |

30 分試して解決しない場合は、状況(スクリーンショット + 試したこと)を課題の
コメントに書いて質問してください。わからないことは恥ずかしいことではありませ
ん。質問した人から伸びます。
