## 設問

**Q1.** `ipconfig` コマンドは何をするためのものか。

- A. PC の電源を切る
- B. PC のネットワーク設定を表示する
- C. インターネットの回線速度を測定する
- D. ファイルを検索する

**Q2.** IPv4 アドレスは、住所システムのたとえで言うと何にあたるか。

- A. 町の出口
- B. 町内の範囲を区切る境界線
- C. あなたの家の住所
- D. 郵便配達員

**Q3.** サブネット マスクの役割として正しいものはどれか。

- A. あなたの家の住所を表す
- B. どこまでが同じ町内かを区切る境界線
- C. 町の外に出る出口
- D. 通信の速度を上げる

**Q4.** デフォルト ゲートウェイの役割として正しいものはどれか。

- A. 同じ町内の相手だけに使う住所
- B. 町の外へ出るときに必ず通る出口
- C. パスワードを保管する場所
- D. データを長期間保存する場所

**Q5.** `255.255.255.0` のような値が表示されるのはどの項目か。

- A. IPv4 アドレス
- B. サブネット マスク
- C. デフォルト ゲートウェイ
- D. コンピュータ名

**Q6.** `ping` を打って `Reply from ...` が返ってきた場合、何を意味するか。

- A. 相手も、そこまでの道もつながっている(成功)
- B. 相手の電源が切れている
- C. 通信が完全に失敗した
- D. IP アドレスの設定が間違っている

**Q7.** `ping` を打って `Request timed out.` が表示された場合、何を意味する
か。

- A. 通信が成功した
- B. 返事が返ってこなかった(どこかで途切れている)
- C. 相手の PC が起動中である
- D. サブネット マスクが正しく設定されている

**Q8.** Packet Tracer とはどのようなソフトか。

- A. 文章を作成するためのソフト
- B. ネットワーク機器を模擬して組み立て・通信を試せるシミュレータ
- C. 画像を加工するソフト
- D. メールを送受信するソフト

**Q9.** Packet Tracer で機器を配置するときの正しい操作はどれか。

- A. 部品箱から機器をワークスペースへドラッグして置く
- B. コマンドを打って機器を呼び出す
- C. 機器は自動的に配置される
- D. 機器の配置はできない

**Q10.** Packet Tracer で PC 2 台が通信するために、各 PC に設定が必要なもの
はどれか。

- A. パスワードとユーザー名
- B. IP アドレスとサブネット マスク
- C. 壁紙とアイコンの位置
- D. インストールした年月日

**Q11.**【P1 復習】コンピュータの三大要素の組み合わせとして正しいものはど
れか。

- A. CPU・メモリ・ストレージ
- B. キーボード・マウス・モニタ
- C. Wi-Fi・Bluetooth・有線 LAN
- D. ブラウザ・OS・アプリ

**Q12.**【P2 復習】コンピュータが情報を 0 と 1(2 進数)で扱う理由として正
しいものはどれか。

- A. 0 と 1 のほうが人間にとって読みやすいから
- B. 電気信号のオン・オフの 2 つの状態だけで正確に区別できるから
- C. 0 と 1 しか計算できない決まりがあるから
- D. 昔からの習慣で変える必要がないから

**Q13.**【P3 復習】Web ページが表示されるまでのおおまかな流れとして正しい
ものはどれか。

- A. ブラウザがサーバにお願いを送り、サーバがデータを送り返す
- B. サーバがブラウザに一方的にデータを送りつけるだけ
- C. ブラウザ同士が直接データをやり取りする

**Q14.**【P4 復習】研修での基本マナーとして正しいものはどれか。

- A. 分からないことがあったら、調べずにすぐ質問してよい

- D. 提出物はスクリーンショットではなく口頭で伝える

**Q15.**(記述)自分の PC で `ipconfig` を実行して分かったことを 1 つ書いて

ドレス・サブネット マスク・デフォルト ゲートウェイの 3 つが何を表すかを、自

## 解答・解説(翌朝公開・講師用)

| 問 | 解答 | 解説 |
|---|---|---|
| Q1 | B | `ipconfig` は、PC が今ネットにどうつながっているか(IPv4 アドレスなど)を表示するコマンド |
| Q2 | C | IPv4 アドレスは、ネット上で自分の PC を一意に指す番地であり、住所のたとえで言えば「あなたの家の住所」にあたる |
| Q3 | B | サブネット マスクは、同じ町内(直接手紙を渡せる範囲)かどうかを区切る境界線の役割を持つ |
| Q4 | B | デフォルト ゲートウェイは、町内にいない相手あての手紙を最初に預ける「町の出口」であり、多くは家庭用ルータが担う |
| Q5 | B | `255.255.255.0` のような形式で表示されるのはサブネット マスク。IPv4 アドレスやデフォルト ゲートウェイは通常こうした値にならない |
| Q6 | A | `Reply from ...` は、山に向かって声を出したらやまびこが返ってきた状態にあたり、相手も道もつながっていることを意味する |
| Q7 | B | `Request timed out.` は返事が返らなかった合図で、相手がいない・電源が切れている・道が切れているといった原因が考えられる |
| Q8 | B | Packet Tracer は、本物の機器を買わずに、画面上でネットワーク機器を組み立てて通信を試せる無料のシミュレータ |
| Q9 | A | 機器は左下の部品箱から、中央のワークスペースへドラッグして配置する |
| Q10 | B | PC 同士が通信するには、それぞれに IP アドレスとサブネット マスクが正しく設定されている必要がある |
| Q11 | A | コンピュータの三大要素は、計算を行う CPU、作業中のデータを一時的に置くメモリ、データを長期間保存するストレージ |
| Q12 | B | コンピュータは電気信号のオン・オフという 2 つの状態しか正確に区別できないため、それを 0 と 1 で表している |
| Q13 | A | ブラウザ(窓口)がサーバ(会社)にお願いの手紙を送り、サーバがページのデータを送り返す、という往復でページが表示される |
| Q14 | B | 研修の基本マナーは、自分でまず 15〜30 分調べる「30 分ルール」と、状況・試したこと・スクショの 3 点セットで質問すること |
| Q15 | 例 | 「ipconfig を実行すると、自分の PC の IPv4 アドレスが分かった」等の具体的な気づき、または「IPv4 アドレスは自分の家の住所、サブネット マスクは町内の範囲を区切る境界線、デフォルト ゲートウェイは町の外へ出るときに通る出口」のように、3 つの役割の違いに触れていれば正解 |

**採点**: Q1〜Q14 は 1 問 6 点(計 84 点)、Q15 は記述式で 16 点、合計 100
点満点。70 点未満は翌朝解き直し。Q15 は完璧な文章でなくても、ipconfig で気
づいたことが具体的に書けているか、または IPv4 アドレス・サブネット マス
ク・デフォルト ゲートウェイの役割の違いが自分の言葉で説明できていれば満点
とする。

**70 点未満だった人へ**: きょうは初めて触るコマンドや画面が多く、間違える
のは当然です。翌朝、解答・解説を読み直してから同じ問題を解き直してくださ
い。解き直しで合格ラインに届けば、それで大丈夫です。Week0 はここで一区切
りです。分からなかったところは誤答ノートに残し、本編が始まってからも見返
してください。
