## 設問

**Q1.** インターネットの説明として正しいものはどれか。

- A. ある 1 つの会社が管理する巨大なコンピュータのこと
- B. 世界中の機器を電線や電波でつないだ網で、中心となる 1 か所は存在しない
- C. 政府だけが使える専用の回線のこと
- D. 個人のパソコン同士だけをつなぐ仕組みのこと

**Q2.** 自分のパソコンやスマホで Web ページを見ているとき、その機器はどちら
の役割を果たしているか。

- A. サーバ
- B. クライアント
- C. どちらでもない
- D. サーバとクライアントの両方を常に同時に果たす

**Q3.** `https://www.example.com/page` のうち、`www.example.com` の部分が
表しているものはどれか。

- A. 通信の約束事
- B. 相手(サーバ)の名前
- C. ページ内のどの部屋か
- D. 送られてくるデータの中身

**Q4.** DNS の役割として正しいものはどれか。

- A. 画像をきれいに表示する
- B. 名前(ドメイン名)を宛先の番号(IP アドレス)に変換する
- C. 通信の内容を暗号化する
- D. Wi-Fi のパスワードを管理する

**Q5.** IP アドレスの説明として正しいものはどれか。

- A. Web ページのタイトルのこと
- B. ネット上の住所を表す番号のこと
- C. ページのデザインを決める指定のこと
- D. パスワードの一種のこと

**Q6.** ブラウザの役割として正しいものはどれか。

- A. データを長期間保管しておく倉庫の役割
- B. サーバにお願いを送り、返ってきたデータを画面に表示する窓口の役割
- C. 電波を発信する基地局の役割
- D. Wi-Fi のパスワードを作る係の役割

**Q7.** 家庭用ルータの役割として正しいものはどれか。

- A. インターネット上のページの中身を作る
- B. 家の機器とインターネットをつなぐ玄関であり、データの仕分け係
- C. 動画データを長期間保存しておく場所
- D. パソコンの中で計算を行う部品

**Q8.** 有線と Wi-Fi の違いとして正しいものはどれか。

- A. 有線は LAN ケーブルでつなぐ、Wi-Fi は電波でつなぐ
- B. 有線も Wi-Fi も、仕組みとしてはまったく同じものである
- C. 有線は電波を使い、Wi-Fi はケーブルを使う
- D. Wi-Fi は家の外でしか使うことができない

**Q9.** スマホの通信方法の組み合わせとして正しいものはどれか。

- A. Wi-Fi と有線
- B. Wi-Fi とモバイル通信(4G/5G)
- C. 赤外線と Bluetooth
- D. 有線と Bluetooth

**Q10.**(記述)URL を打ってからページが表示されるまでの流れを、「郵便を送
る」のたとえ(窓口・電話帳・相手の会社・手紙)を使って、自分の言葉で説明せ
よ。

## 解答・解説(翌朝公開・講師用)

| 問 | 解答 | 解説 |
|---|---|---|
| Q1 | B | インターネットは無数の機器がバケツリレーのようにデータを渡し合う網であり、中心となる本社のような場所は存在しない |
| Q2 | B | 自分の PC やスマホは、ページを見に行く側であるクライアント。ページの中身を持っているのはサーバ側 |
| Q3 | B | `https://` は通信の約束事、`www.example.com` は相手(サーバ)の名前、`/page` はその中のどの部屋かを表す |
| Q4 | B | DNS は「名前 → 番号」を調べる電話帳の役割を持ち、`example.com` のような名前を IP アドレスという番号に変換する |
| Q5 | B | IP アドレスは `192.0.2.1` のような数字で表される、ネット上の住所(宛先)を表す番号 |
| Q6 | B | ブラウザは郵便局の窓口にあたり、お願いの手紙を送り、返ってきたデータを画面に表示する役割を持つ |
| Q7 | B | 家庭用ルータは家の玄関であり、外から来た手紙をどの機器宛てか振り分け、家から出す手紙を外へ送り出す仕分け係 |
| Q8 | A | 有線は LAN ケーブルで直接つなぐため安定・速い。Wi-Fi は電波で無線につなぐため線が不要だが不安定なこともある |
| Q9 | B | スマホは家では Wi-Fi、外では基地局を使うモバイル通信(4G/5G)につながり、多くは自動で切り替わっている |
| Q10 | 例 | 「ブラウザ(窓口)は、まず DNS(電話帳)に相手の会社の番号(IP アドレス)を尋ねる。番号が分かったら、窓口はそのページをくださいという手紙を相手の会社(サーバ)に送る。会社は注文どおりのデータを手紙にして送り返し、それが手元で組み立てられて画面に表示される」等、窓口=ブラウザ・電話帳=DNS・会社=サーバ・手紙=データ、の対応と、名前を番号に変えてから送っている、という順序に触れていれば正解 |

**採点**: 1 問 10 点、70 点未満は翌朝解き直し。Q10 は選択式のような部分点計算
ではなく、「窓口(ブラウザ)が電話帳(DNS)で会社(サーバ)の番号を調べてから
手紙(お願い)を送る」「会社が手紙(データ)を送り返す」という順序が説明でき
ていれば 10 点とする。

**70 点未満だった人へ**: 今日は初めて聞く言葉が多く、間違えるのは当然です。
翌朝、解答・解説を読み直してから同じ問題を解き直してください。解き直しで合格
ラインに届けば、それで大丈夫です。
