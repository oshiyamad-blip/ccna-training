## 設問

**Q1.** コンピュータが行っている基本の流れとして正しい順番はどれか。

- A. 出力 → 処理 → 入力
- B. 入力 → 処理 → 出力
- C. 処理 → 入力 → 出力
- D. 入力 → 出力 → 処理

**Q2.** 次のうち、入力装置はどれか。

- A. 画面(ディスプレイ)
- B. スピーカー
- C. キーボード
- D. プリンタ

**Q3.** CPU の役割として正しいものはどれか。

- A. データを長期間保管しておく
- B. 計算や判断を実際に行う
- C. 画面に文字や画像を表示する
- D. 音声を録音する

**Q4.** メモリ(RAM)の役割として正しいものはどれか。

- A. いま作業中のデータを広げておく
- B. 写真や書類を長期間保管しておく
- C. 電源を切っても中身が消えない場所である
- D. ファイルの種類を表す名札を管理する

**Q5.** ストレージ(SSD/HDD)の役割として正しいものはどれか。

- A. 計算や判断を実際に行う
- B. いま作業中のデータだけを一時的に置く
- C. 写真・書類・アプリなどを保管しておく
- D. 電源を切ると中身が消える

**Q6.** 電源を切ったときに中身が消えるのはどちらか。

- A. メモリ(RAM)
- B. ストレージ(SSD/HDD)
- C. どちらも消える
- D. どちらも消えない

**Q7.** 次のうち、OS はどれか。

- A. Word
- B. ブラウザ
- C. Windows
- D. Excel

**Q8.** ファイルとフォルダの説明として正しいものはどれか。

- A. ファイルはフォルダを入れてまとめる入れ物である
- B. フォルダはファイルを入れてまとめる入れ物である
- C. ファイルとフォルダは同じものを指す
- D. フォルダの中にフォルダを作ることはできない

**Q9.** 拡張子「.jpg」が表すファイルの種類はどれか。

- A. 文書(テキスト)
- B. 画像
- C. Excel の表
- D. PDF 文書

**Q10.**(記述)メモリとストレージの違いを、「作業机」と「本棚」のたとえを
使って、電源を切ったときにそれぞれどうなるかに触れながら、自分の言葉で説明
せよ。

## 解答・解説(翌朝公開・講師用)

| 問 | 解答 | 解説 |
|---|---|---|
| Q1 | B | コンピュータは「情報を受け取る(入力)→処理する→結果を出す(出力)」の順に動く |
| Q2 | C | キーボードは情報をコンピュータに入れる入力装置。画面・スピーカー・プリンタはいずれも結果を外に出す出力装置 |
| Q3 | B | CPU は作業机のたとえでいう「作業する人」にあたり、計算や判断を実際に行う部品 |
| Q4 | A | メモリは作業机のたとえでいう「机の広さ」。いま作業中のデータを広げておく場所で、電源を切ると中身が消える |
| Q5 | C | ストレージは作業机のたとえでいう「本棚」。写真・書類・アプリなどを電源を切っても消えない状態で保管する |
| Q6 | A | メモリは電源を切ると中身が消える。ストレージは電源を切っても中身が残る |
| Q7 | C | Windows は電源を入れて最初に動き出す土台のソフト(OS)。Word・Excel・ブラウザはいずれも OS の上で動くアプリ |
| Q8 | B | フォルダはファイルをまとめて入れる入れ物で、フォルダの中にさらにフォルダを作る(入れ子にする)こともできる |
| Q9 | B | 拡張子はファイルの種類を表す名札。「.jpg」は画像ファイルを表す |
| Q10 | 例 | 「メモリは作業机のようなもので、いま使っているデータを広げておく場所であり、電源を切ると机の上が片付くように中身が消える。ストレージは本棚のようなもので、写真や書類をしまっておく場所であり、電源を切っても本棚の中身が残るようにデータは消えない」等、机=メモリ・消える、本棚=ストレージ・残る、の対応に触れていれば正解 |

**採点**: 1 問 10 点、70 点未満は翌朝解き直し。Q10 は選択式のような部分点計算
ではなく、「机=メモリ=電源を切ると消える」「本棚=ストレージ=電源を切っても
残る」の対応が説明できていれば 10 点とする。

**70 点未満だった人へ**: 今日は初めて聞く言葉ばかりで、間違えるのは当然です。
翌朝、解答・解説を読み直してから同じ問題を解き直してください。解き直しで
合格ラインに届けば、それで大丈夫です。
