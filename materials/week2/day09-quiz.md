## 設問

**Q1.** ブリッジングループが発生した場合に起こる現象として、最も適切なものはどれか。

- A. IP パケットの TTL が 0 になり、送信元へ破棄通知が返る
- B. スイッチが MAC アドレステーブルをすべて自動的にクリアする
- C. BUM フレームがループを回り続け、帯域を消費するブロードキャストストームが発生する
- D. ルータがルーティングループを検出し、ICMP リダイレクトを送信する

**Q2.** BID（Bridge ID）とルートブリッジ選出に関する記述として正しいものはどれか。

- A. BID は 6 バイトの MAC アドレスのみで構成される
- B. プライオリティが同値の場合、MAC アドレスが最も小さいスイッチがルートブリッジに選ばれる
- C. BID は VLAN ごとに固定された 8 バイトの乱数値である
- D. プライオリティが最大のスイッチがルートブリッジに選ばれる

**Q3.** ブリッジプライオリティに関する記述として正しいものはどれか。

- A. 既定値は 0 であり、1 刻みで設定できる
- B. 既定値は 4096 であり、256 刻みで設定できる
- C. 既定値は 61440 であり、任意の整数値で設定できる
- D. 既定値は 32768 であり、4096 の倍数でのみ設定できる

**Q4.** 非ルートブリッジのスイッチにおいて、ルートブリッジへの最小ルートパスコストを
持つポートのロールはどれか。

- A. ルートポート
- B. 指定ポート
- C. 非指定ポート（ブロッキング）
- D. バックアップポート

**Q5.**（出力読解）SW2（ルートブリッジではない）で `show spanning-tree vlan 10`
を実行したところ、Gi0/1 と Gi0/2 の 2 つのポートが候補として表示された。
Gi0/1 経由でルートブリッジに至る経路は 1 Gbps 区間のみ（コスト 4）、
Gi0/2 経由でルートブリッジに至る経路は 1 Gbps 区間 + 100 Mbps 区間
（コスト 4 + 19 = 23）を通る。この場合、SW2 の Gi0/1・Gi0/2 の Role/Sts 欄は
それぞれどう表示されるべきか、最も適切なものはどれか。

- A. Gi0/1: Root/FWD、Gi0/2: Desg/FWD（ルートパスコストが最小の Gi0/1 がルートポートになる。Gi0/2 は対向スイッチ自身のルートまでのコスト（19）より SW2 の広告コスト（4）の方が小さいため、そのセグメントでは SW2 側が指定ポートになる）
- B. Gi0/1: Altn/BLK、Gi0/2: Root/FWD（ルートパスコストが大きい経路が優先される）
- C. Gi0/1・Gi0/2 とも Desg/FWD になる（コストに関わらず両方転送する）
- D. Gi0/1・Gi0/2 とも Root にはなれず、両方 Blocking になる

**Q6.** 802.1D のポートステートの遷移順序として正しいものはどれか。

- A. Blocking → Learning → Listening → Forwarding
- B. Listening → Blocking → Learning → Forwarding
- C. Blocking → Listening → Learning → Forwarding
- D. Learning → Listening → Blocking → Forwarding

**Q7.** RSTP（802.1w）のポートロールに関する記述として正しいものはどれか。

- A. RSTP にはルートポートと指定ポートしか存在しない
- B. Backup ロールは、ルートポートの直接のバックアップとして動作する
- C. Alternate ロールは、指定ポートのバックアップとして同一セグメントに複数ポートがある場合に生じる
- D. Alternate ロールは、ルートポートの代替経路として機能し、障害時にすぐ切り替えられるよう待機する

**Q8.** PortFast と BPDU Guard の組み合わせに関する記述として正しいものはどれか。

- A. PortFast を設定したポートで BPDU を受信すると、BPDU Guard によりそのポートは err-disable になる
- B. PortFast はトランクポートに設定することが推奨される
- C. BPDU Guard は BPDU を受信しても常にポートを転送状態のまま維持する
- D. PortFast を設定すると Max Age タイマが 2 倍になる

**Q9.** LACP のモード組み合わせに関する記述として正しいものはどれか。

- A. passive と passive の組み合わせで EtherChannel が成立する
- B. active と active の組み合わせでは EtherChannel は成立しない
- C. LACP は Cisco 独自のプロトコルであり、他ベンダーとは相互接続できない
- D. active と passive の組み合わせで EtherChannel が成立する

**Q10.**（記述）次の 3 台のスイッチが冗長トライアングル状に接続されている
（VLAN 1、拡張システム ID を考慮しない簡略化した条件とする）。

| スイッチ | ブリッジプライオリティ | MAC アドレス |
|---|---|---|
| SW-A | 32768 | 0011.1111.1111 |
| SW-B | 24576 | 0022.2222.2222 |
| SW-C | 32768 | 0033.3333.3333 |

(1) どのスイッチがルートブリッジに選出されるか、根拠とともに説明せよ。
(2) SW-A を VLAN 1 のルートブリッジに変更したい場合、SW-A に投入すべきコマンドを
1 つ記述せよ。

## 解答・解説（翌日公開・講師用）

| 問 | 解答 | 解説 |
|---|---|---|
| Q1 | C | ブリッジングループでは BUM（ブロードキャスト/未知ユニキャスト/マルチキャスト）フレームがループを回り続け、帯域を急速に消費するブロードキャストストームが発生する。イーサネットフレームには TTL がなく自然消滅しない |
| Q2 | B | ルートブリッジ選出はまず BID（プライオリティ + MAC アドレス）が最小のスイッチを選ぶ。プライオリティが同値の場合は MAC アドレスが最も小さいスイッチが選ばれる |
| Q3 | D | ブリッジプライオリティの既定値は 32768 で、4096 の倍数（0, 4096, 8192, …, 61440）でのみ設定できる |
| Q4 | A | ルートブリッジへの最小ルートパスコストを持つポートがルートポートであり、各スイッチにつき必ず 1 つだけ選ばれる |
| Q5 | A | ルートパスコストは Gi0/1 が 4、Gi0/2 経由が 4 + 19 = 23（SW2-対向スイッチ間のリンクコスト4 + 対向スイッチ自身のルートまでのコスト19）。コストが小さい Gi0/1 がルートポート（Root/FWD）になる。Gi0/2 のセグメントでは、SW2 自身がルートまで広告するコスト（4）が対向スイッチの広告コスト（19）より小さいため、SW2 側が指定ポートになりブロッキングされない（Desg/FWD）。ブロッキングされるのは対向スイッチ側のポートである（講義2章の SW2/SW3 の数値例と同じ構造） |
| Q6 | C | 802.1D のステートは Disabled を除くと Blocking → Listening → Learning → Forwarding の順に遷移する |
| Q7 | D | Alternate ロールはルートポートのバックアップであり、障害時にすぐ切り替えられるよう待機する。指定ポートのバックアップは Backup ロールである |
| Q8 | A | BPDU Guard は、PortFast（エッジポート）で本来届かないはずの BPDU を受信すると、そのポートを err-disable にしてループを防ぐ |
| Q9 | D | LACP は active-active・active-passive の組み合わせで成立する。passive-passive はどちらも相手の開始を待つため成立しない |
| Q10 | 例 | (1) SW-B が選出される。理由: 3 台の中でブリッジプライオリティが最小（24576）であり、プライオリティが最も優先されるため、MAC アドレスの比較を待たずに SW-B が最小の BID を持つ。 (2) `SW-A(config)# spanning-tree vlan 1 root primary` |

**採点**: 1 問 10 点、70 点未満は翌朝再テスト。Q10 は (1)(2) 双方の趣旨が合っていれば
10 点（プライオリティ最小のスイッチが優先される点、および `root primary` コマンドに
触れていれば正解とする）。
