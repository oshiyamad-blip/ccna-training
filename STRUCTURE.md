## 2. Week0 プレコース（`materials/week0/` — IT 超入門・5 パート）

各パート = **講義（lecture）＋ 実習（work）＋ 小テスト（quiz）** の 3 点セット。

### P1 コンピュータの基礎（p01-lecture）
- 第1章 コンピュータって、そもそも何をしている機械?
- 第2章 ハードウェアの 3 要素（CPU / メモリ / ストレージ）
- 第3章 OS とアプリ — 土台と道具
- 第4章 ファイルとフォルダ — 置き場所を住所で考える
- 第5章 拡張子 — ファイルの種類を表す名札

### P2 数の基礎（p02-lecture）
- 第1章 なぜコンピュータは 0 と 1 しか使えないのか
- 第2章 10進数をあらためて分解する
- 第3章 2進数（本日の最重要）
- 第4章 16進数と MAC アドレス
- 第5章 bit と Byte、K・M・G の単位

### P3 インターネットの基礎（p03-lecture）
- 第1章 インターネットって、そもそも何?
- 第2章 URL を打ってから画面が出るまで
- 第3章 家がインターネットにつながる仕組み
- 第4章 スマホと、外出先での通信
- 第5章 今日の言葉と本編への橋渡し

### P4 仕事のIT基礎（p04-lecture）
- 第1章 Backlog って何?（画面ツアー）
- 第2章 課題を1件開いてみる（状態・コメント・添付）
- 第3章 Markdown で読みやすく書く
- 第4章 スクリーンショットを撮って貼る
- 第5章 分からないときの動き方（検索＋質問テンプレート）
- 第6章 仕事のセキュリティの基礎（パスワード）

### P5 ネットワークに触れる（p05-lecture）
- 第1章 きょうやること（Week0 の総仕上げ）
- 第2章 自分の PC の設定を見る（ipconfig）
- 第3章 3 つの数字の意味（住所・範囲・出口）
- 第4章 ping
- 第5章 Packet Tracer とは
- 第6章 本編への橋渡し

> 各パートに対応する **実習手順書（p0N-work）** と **小テスト（p0N-quiz）** あり。

## 3. Week1〜4 本編（`materials/week1〜4/` — Day01〜20）

各 Day = **講義（day##-lecture）＋ ラボ（day##-lab）＋ 小テスト（day##-quiz）**。
各講義冒頭に「ウォームアップ（朝の想起クイズ）」を配置（Day02 以降）。

### Week1 ネットワーク基礎（ドメイン1）
- **Day01** ネットワークの全体像と OSI / TCP-IP モデル
  （ネットワークとは／トポロジ／OSI・TCP-IP／カプセル化と PDU）
- **Day02** Cisco IOS の基本操作とデバイス初期設定
  （CLI アクセス／コマンドモード／基本設定／管理 IP／SSH／設定保存）
- **Day03** IPv4 アドレッシングとサブネット化
  （構造／クラス・特殊アドレス／RFC1918・NAT 前提／マスク・プレフィックス／計算／VLSM）
- **Day04** IPv6 アドレッシング
  （必要性・構造／表記・省略／アドレスタイプ／EUI-64／設定・SLAAC・DHCPv6／NDP）
- **Day05** TCP/UDP・スイッチング動作・物理層
  （トランスポート層／3wayハンドシェイク・ポート／MAC 学習／ケーブル・速度・デュプレックス／PoE／仮想化）

### Week2 ネットワークアクセス（ドメイン2）
- **Day06** VLAN の基礎
  （ブロードキャストドメイン分割／種類・デフォルト VLAN／作成・命名／アクセスポート／確認・TS）
- **Day07** トランクと VLAN 設計
  （802.1Q タギング／ネイティブ VLAN／許可 VLAN 制御／DTP／VTP／検証・TS）
- **Day08** VLAN 間ルーティング
  （必要性／Router-on-a-Stick／L3 スイッチと SVI／ルーテッドポート／方式比較／確認・TS）
- **Day09** STP と EtherChannel
  （ブリッジングループ／ルートブリッジ選出・BID／ポートロール・ステート・タイマ／RSTP・PortFast・BPDU Guard／EtherChannel・LACP）
- **Day10** 無線 LAN と検出プロトコル
  （電波・チャネル・SSID／自律型 AP・Lightweight AP＋WLC／WPA2・WPA3／WLC GUI／CDP・LLDP）

### Week3 IP コネクティビティ + IP サービス（ドメイン3・4）
- **Day11** ルーティングの基礎と静的ルート
  （ルーティングテーブル／経路選択 3 段階／静的ルート／デフォルト・フローティング／IPv6 静的）
- **Day12** OSPFv2（シングルエリア）
  （リンクステート／パケット・ネイバー確立／Router ID・DR/BDR／コスト計算／設定・パッシブ／確認・TS）
- **Day13** OSPF 応用と FHRP
  （ネイバー確立復習／不成立の切り分け／default-information originate／HSRP・VRRP・GLBP／HSRP 詳細）
- **Day14** NAT
  （必要性／inside・outside・local・global／静的 NAT／動的 NAT／PAT・オーバーロード／確認・TS）
- **Day15** IP サービス総合
  （DHCP・DORA／DNS／NTP・stratum／SNMP・Syslog／SSH・Telnet・FTP・TFTP／QoS）

### Week4 セキュリティ + 自動化 + 総まとめ（ドメイン5・6）
- **Day16** セキュリティの概念とデバイス保護
  （基本用語／攻撃の種類／AAA／パスワードポリシー・デバイス保護／セキュリティプログラム・教育）
- **Day17** ACL
  （役割・仕組み／評価ロジック・暗黙 deny／ワイルドカードマスク／標準 ACL／拡張 ACL／VTY 制限・ロギング・TS）
- **Day18** レイヤ2 セキュリティと VPN 概要
  （L2 脅威／ポートセキュリティ／DHCP スヌーピング／DAI／サイト間・リモートアクセス VPN／IPsec）
- **Day19** 自動化とプログラマビリティ
  （SDN・コントローラベース／オーバーレイ・アンダーレイ・ファブリック／Catalyst Center／REST API／JSON／Ansible・Terraform／ネットワーク運用と AI）
- **Day20** 総復習と修了試験対策
  （総復習の進め方・弱点分析／ドメイン1・2 総復習／ドメイン3・4 総復習／ドメイン5・6 総復習／CCNA 受験手続き・当日の流れ／修了後の学習ガイダンス）
